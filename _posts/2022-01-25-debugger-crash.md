- core文件生效
    ```bash
    echo "* soft core unlimited" | sudo dd of=/etc/security/limits.d/core.conf
    #重启 下面这行无法永久修改 重启后总是被重写 以后解决
    sudo sysctl -w kernel.core_pattern=/tmp/core-%e.%p.%h.%t
    ```

- 编辑崩溃程序(a.cpp)
    ```cpp
    #include <iostream>
    
    void test01()
    {
        int *p = nullptr;
        *p = 23;
    }
    
    int main()
    {
        test01();
        return 0;
    }
    ```
- 编译奔溃程序(注意要加-g)
    ```bash
    > g++ -o bin -g a.cpp
    ```
- 执行崩溃程序生成core文件
    ```bash
    > ./bin
    > ls /tmp/ | grep core
    ```
    ```plain
    core-bin.4079.ubuntu.1643104689
    ```
